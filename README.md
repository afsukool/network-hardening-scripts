# Network Hardening and Security Scripts

A collection of PowerShell and Bash scripts designed to secure Windows and Linux environments.

## Includes
- Windows Firewall rule automation
- Linux iptables rules
- Service hardening
- Basic network monitoring commands

## Tools Used
- PowerShell
- Bash / Shell
- netsh and iptables

## Files
- `windows_firewall_rules.ps1`
- `linux_firewall_rules.sh`
